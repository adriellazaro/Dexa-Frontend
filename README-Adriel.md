# Dexa Attendance System - Frontend

Frontend application for attendance system built using React Vite.

## Features

- Login & Register page
- Dashboard (Check-in / Check-out)
- Attendance table (User & Admin view)
- Filter attendance by date
- Profile page (Update user data)
- Role-based UI (Admin & User)

## Tech Stack

- React (Vite)
- Axios
- JWT Decode

## Install Dependencies
npm install

## Backend API
http://localhost:3000

## Notes
Requires backend API to function
JWT token is stored in localStorage
Redirects to login if token is missing
